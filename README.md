# Gestionnaire de villes touristiques

Projet universitaire réalisé en Licence 2 Informatique (Université de Lorraine).

## Description

Application JavaFX permettant de gérer une collection de villes touristiques.

L'utilisateur peut consulter, ajouter, modifier et supprimer des villes, ainsi qu'importer et exporter les données au format JSON.

## Technologies

- Java 21
- JavaFX
- FXML
- MVC
- JSON

## Fonctionnalités

- Affichage de la liste des villes
- Affichage détaillé d'une ville
- Ajout de villes
- Modification de villes
- Suppression de villes
- Recherche
- Tri
- Import JSON
- Export JSON

## Lancement

1. Ouvrir le projet dans IntelliJ IDEA.
2. Utiliser Java 21.
3. Configurer JavaFX.
4. Lancer la classe Main.
